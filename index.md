# 南信州 SNSもみじ湖キャンプ 
    - 箕輪ダムのgoogle ストリートビューをみる
    - もみじ湖（箕輪ダム）キャンプ場をみる
    - 萱野高原へのトレッキングを体験する
    
- 箕輪ダム（みのわダム）は、長野県上伊那郡箕輪町、一級河川・天竜川水系沢川に建設された
ダム。高さ72メートルの重力式コンクリートダムで、洪水調節・不特定利水・上水道を目的とす
る、長野県営の多目的ダムである。ダム湖（人造湖）の名はもみじ湖（もみじこ）という。
